# Designer Skill

## Responsibilities
- **Visuals**: Create stunning, modern UI/UX using Vanilla CSS or Tailwind (if requested).
- **Responsiveness**: Mobile-first approach.
- **Aesthetics**: High-end look (gradients, glassmorphism, proper spacing).

## CSS Architecture
1.  **Variables**: Define a `:root` block with semantic color names (`--color-primary`, `--bg-surface`).
2.  **Reset**: Always include a modern CSS reset.
3.  **Components**: Use classes like `.card`, `.btn`, `.hero` for reusability.

## Output
- Produce `style.css` (for Vanilla) or `tailwind.config.js` + classes.
- Ensure `index.html` structure matches the styling hooks.
- **Review**: If browser tool is available, verify the look and feel.
