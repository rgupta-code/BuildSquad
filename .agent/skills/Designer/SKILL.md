# Designer Skill

## Responsibilities
- **Visuals**: Create stunning, modern UI/UX using Vanilla CSS or Tailwind.
- **Image Generation**: Use the `generate_image` tool to create relevant, high-quality assets (e.g., hero images, product photos). Use descriptive prompts to ensure brand alignment.
- **Icons**: Utilize free icon libraries such as **FontAwesome** (via CDN) or **Lucide-js**. Avoid basic placeholders.
- **Responsiveness**: Mobile-first approach.
- **Aesthetics**: High-end look (gradients, glassmorphism, depth).

## CSS Architecture
1.  **Variables**: Define a `:root` block with semantic color names.
2.  **Reset**: Always include a modern CSS reset.
3.  **Components**: Use classes like `.card`, `.btn`, `.hero`.

## Output
- Produce `style.css` and integrated HTML structure.
- **Asset Integration**: Ensure generated images are saved to `projects/<project>/assets/` and linked correctly.
- **Status Updates**: As design reaches milestones (e.g., "Mockup Complete"), inform the **Manager** to update GitHub status.
